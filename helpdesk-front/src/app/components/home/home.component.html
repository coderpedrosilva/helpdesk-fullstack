Help Desk Projeto Full Stack

Olá! Bem-vindo ao sistema Help Desk, desenvolvido no curso de Desenvolvedor Full Stack. Trata-se de um sistema para gerenciamento de chamados, simulando um simples Help Desk, desenvolvido com Angular 12 para o front-end e Spring Boot 2 para o back-end. O sistema está hospedado na nuvem, com o front-end na Vercel e o back-end na Render. O principal objetivo do projeto é abordar conceitos de aplicações SPA (Single Page Applications), autenticação e autorização com Tokens JWT, entre outros. O código-fonte do projeto pode ser acessado através do link do GitHub no menu lateral.

Sinta-se à vontade para testar o sistema e deixar suas sugestões.
